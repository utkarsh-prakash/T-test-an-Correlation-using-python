# T-test-an-Correlation-using-python

### Basic difference between paired and independent t tests.

<b>Paired T test</b><br>
compare scores on two different variables but for the same group of cases.<br> 
How the same group of people will perform on two different tests.

<b>Independent T test</b><br>
compare scores on the same variable but for two different groups of cases.<br>
How 2 different groups of people will perform on same test.
